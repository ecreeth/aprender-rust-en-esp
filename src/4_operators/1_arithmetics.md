# Operadores Aritméticos 

Los operadores aritméticos se utilizan para realizar operaciones aritméticas.

```rust, editable
fn main() {
    let a = 5;
    let b = 8;
    
    println!("Valor de a: {}, Valor de b: {}", a , b);

    println!("Suma: {}", a + b);
    println!("Resta: {}", a - b);
    println!("Multiplicacion: {}", a * b);
    println!("Division: {}", a / b);
    println!("Modulo: {}", a % b);
}
```
